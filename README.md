# InvestTrack Pro - Professional Portfolio Management

A modern, fully-featured investment portfolio tracker with real-time market data, dark mode, and professional analytics.

## Features

✅ **Real-Time Market Data** - Live prices from Yahoo Finance  
✅ **Portfolio Management** - Track stocks, ETFs, and bonds  
✅ **Performance Analytics** - Visualize portfolio allocation  
✅ **Dark Mode** - Professional dark theme  
✅ **CSV Export** - Download your portfolio data  
✅ **Cost Basis Tracking** - Monitor gains and losses  
✅ **Responsive Design** - Works on mobile, tablet, desktop  
✅ **Zero Configuration** - No backend needed  

## Quick Start

1. Download `index.html`
2. Open in your web browser
3. Start adding your investments

## Deployment Options

### Option A: Deploy to Netlify (Recommended)

1. Go to [netlify.com](https://netlify.com) and sign up (free)
2. Click "Add new site" → "Deploy manually"
3. Drag `index.html` onto Netlify
4. Your site goes live instantly!

### Option B: Deploy to GitHub Pages

1. Create a GitHub account at [github.com](https://github.com)
2. Create a new repository named `investtrack-pro`
3. Upload `index.html` to your repository
4. Go to Settings → Pages
5. Select main branch as source
6. Your site will be at: `username.github.io/investtrack-pro`

### Option C: Connect GitHub to Netlify (Best for Updates)

1. Push code to GitHub
2. Go to [netlify.com](https://netlify.com)
3. Click "New site from Git"
4. Connect your GitHub account
5. Select your repository
6. Netlify auto-deploys on every push!

## How to Use

### Add an Investment
1. Go to "My Holdings" tab
2. Enter ticker symbol (e.g., VFV, AAPL)
3. Enter number of shares
4. Optionally enter purchase price to track gains/losses
5. Click "Add to Portfolio"

### View Analytics
1. Click "Analytics" tab
2. See portfolio summary and allocation chart
3. Track total value and returns

### Export Your Data
1. Click "Export CSV" button
2. Download your portfolio as CSV file
3. Open in Excel or Google Sheets

### Toggle Dark Mode
1. Click "Dark Mode" button
2. Theme saves automatically

## Supported Tickers

Works with any ticker from:

- **Canadian Stocks**: RCI, BCE, TD, etc.
- **ETFs**: VFV, XIC, XUS, SPY, etc.
- **Bonds**: BND, AGG, etc.
- **International**: Any ticker on Yahoo Finance

## Technical Details

- **No Backend Required** - Runs entirely in your browser
- **Real-Time Data** - Fetches from Yahoo Finance API
- **Local Storage** - Data saved in browser (never sent anywhere)
- **No Accounts** - No login required
- **Works Offline** - Cached data available without internet
